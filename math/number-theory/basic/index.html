    C 语言的整数除法和取模运算¶

    在 C 语言中，整除除法和取模运算，与数学上习惯的取模和除法不一致。

    C 语言中的整数除法不是一律向下取整，而是向 0 取整。取模也同理。

    因此，一个负数模一个正数，结果会得到一个负数，这与数学上的习惯不同。

    C 语言对于除数或模数为负数的行为是未定义行为，结果未定。这一点也要特别注意。
